Setting up a static website with nginx on Ubuntu 20. Part two.

1. First, let's download some certificates. For that, we will need to download a piece of software called certbot.

sudo add-apt-repository ppa:certbot/certbot

sudo apt-get install python-certbot-nginx

2. To get the certificates, we need a domain name.To get one from compute canada, type:

host *your server's public IP address*

This command will give you the domain name.

3. Now we are ready to get certificates.

sudo certbot --nginx -d *your domain name*

Follow the installation instructions. Choose option 2 when asked about redirection, this way, all http pages will automatically be converted to https.
