Setting up a static website with nginx on Ubuntu 20.

1. Start by updating Ubuntu.

sudo apt-get update

sudo apt-get upgrade

2. Install nginx.

sudo apt-get install nginx

3. Open the ports so that the user can get response from the server. Another way to put it: we are telling the firewall that it is OK to get http/https requests from the user (us) with a specific IP adress (your IP adress). In order to do this, go to your server and manage securuty rules (add an http and an https rule). Now, you should be able to see nginx welcome page when your type your cloud's external IP adress in the search line.

4. Create a directory in which you are going to store your code.

sudo mkdir /var/www/files-dir

5. Now we should configure nginx to serve pages that are situated in our newly created folder. Go to nginx sites-available folder, make a copy of the default file and, in the new copy change the root from /var/www/html to /var/www/files-dir.

cd /etc/nginx/sites-available

sudo cp default mydefault

6. Make an external link of a newly created file to the /sites-enabed directory. Also, delete an existing default file in the sites-enabled directory (do not confuse with sites-available.

ln -s /etc/nginx/sites-available/mydefault /etc/nginx/sites-enabled/mydefault

sudo rm /etc/nginx/sites-enabled/default

7. Restart nginx.

sudo systemctl restart nginx

If problems occur, log them.

sudo systemctl status nginx

You should see your html page now if you type your domain name in the browser. of this tutorial on how to convert your site protocol to https. On this page you can see troubleshooting of some error I have encounterd.
